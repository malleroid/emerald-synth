# Vivaldi Theme — Emerald Synth

## Theme colors

| Setting       | Value     |
|---------------|-----------|
| Background    | `#0d0221` |
| Foreground    | `#f0e6ff` |
| Highlight     | `#00ffcc` |
| Accent        | `#00d4aa` |

## Settings

- Accent from page: off
- Accent on window: off
- Accent saturation limit: 100%
- Transparent tab bar: off
- Transparent background tabs: on
- Blur: 0
- Corner radius: 7

## Upload description (for themes.vivaldi.net)

> A synthwave-inspired dark theme with deep purple backgrounds and cyan-green accents. Features a subtle radial glow background image with teal and pink light sources. Designed for comfortable extended use with high contrast between UI elements.

## Install

1. Download `theme/` directory as a zip (or export from Vivaldi)
2. In Vivaldi: Settings > Themes > Import theme
3. Select the zip file
