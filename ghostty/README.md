# Emerald Synth for Ghostty

A synthwave-inspired dark theme for [Ghostty](https://ghostty.org) with deep purple backgrounds and emerald-green foreground text.

## Install

```sh
ln -s /path/to/emerald-synth/ghostty/emerald-synth ~/.config/ghostty/themes/emerald-synth
```

Then add to `~/.config/ghostty/config`:

```
theme = emerald-synth
```
